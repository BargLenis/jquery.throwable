Welcome to this page about JQuery.throwable plugin

This page describe the JQuery.throwable plugin and how to use it

a simple use

$("#htmlElement").throwable(); // this will make the html element with the id=htmlElment throwable like a physical object

JQuery.throwable has other options you can overwrite
